# Engineering skills

- [`bug-to-pr`](./bug-to-pr/SKILL.md) — diagnose, test, fix, review, and open a pull request for a bug.
- [`feature-to-pr`](./feature-to-pr/SKILL.md) — implement a feature as tested tracer bullets and open a pull request.
- [`slack-bug-to-pr`](./slack-bug-to-pr/SKILL.md) — adapt a Slack bug thread into the bug-to-PR workflow.
- [`slack-feature-to-pr`](./slack-feature-to-pr/SKILL.md) — adapt a Slack feature thread into the feature-to-PR workflow.
