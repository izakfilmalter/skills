# feature-to-pr

```bash
npx skills add izakfilmalter/skills --skill feature-to-pr bug-to-pr request-to-pr
```

[`feature-to-pr`](../../skills/engineering/feature-to-pr/SKILL.md) turns a requested behavior change into a ready-for-review GitHub pull request. It maps substantial acceptance behaviors to tested vertical tracer bullets, requires end-to-end evidence for substantial frontend work, and reviews substantial results for acceptance coverage and minimality.

The skill composes [`request-to-pr`](../references/request-to-pr.md) for repository intake, project evidence, review remediation, pull-request creation, and final-response rules.
