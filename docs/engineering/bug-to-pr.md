# bug-to-pr

```bash
npx skills add izakfilmalter/skills --skill bug-to-pr feature-to-pr request-to-pr
```

[`bug-to-pr`](../../skills/engineering/bug-to-pr/SKILL.md) turns a bug report into a ready-for-review GitHub pull request. It distinguishes promised behavior from new behavior, researches the execution path, proves substantial bugs with a failing regression test, fixes them through red-green tracer bullets, validates the integrated behavior, and runs correctness and minimality reviews for substantial work.

The skill composes [`request-to-pr`](../references/request-to-pr.md) for repository intake, project evidence, review remediation, pull-request creation, and final-response rules.
