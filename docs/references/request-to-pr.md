[`request-to-pr`](../../skills/references/request-to-pr/SKILL.md) is the model-invoked workflow primitive shared by `bug-to-pr` and `feature-to-pr`. It owns request intake, branch isolation, project evidence, worker boundaries, review remediation, GitHub pull-request creation, and direct-versus-parent response behavior.

Install it alongside any task-facing skill that references `/request-to-pr`.
